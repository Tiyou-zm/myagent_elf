# 绯铃待机眨眼任务单

## 用途

这份文档只用于让出图 agent 产出绯铃的第一组最小眨眼素材。

当前目标非常明确：

- 只做 `idle blink`
- 不做完整逐帧全身动画系统
- 不扩展到 `happy_soft`、`thinking`、`confused`、`smug`

## 当前基准素材

请严格基于以下母版继续出图：

- `assets/characters/feiling/states/feiling_idle_v1.jpg`

角色必须保持不变：

- 同一头身比
- 同一脸型
- 同一眼睛位置
- 同一发型
- 同一狐耳位置
- 同一服装
- 同一尾巴结构
- 同一构图
- 同一角色大小

重点：

- 不要把它理解成“重新画一张新立绘”
- 这是在现有待机图基础上，只补一组眨眼帧

## 任务范围

只输出两张补充图：

1. `blink_half`
2. `blink_closed`

如果平台稳定，也可以额外给：

3. `blink_half_return`

但最低只需要前两张就够程序先接了。

## 动作要求

眨眼要自然、轻、小：

- 睁眼 -> 半闭 -> 闭眼 -> 半闭 -> 睁眼
- 重点是“安静活着”的感觉
- 不要做成卖萌表情包
- 不要加大幅微笑
- 不要加头部夸张摆动
- 不要顺便改嘴型

一句话：

这是待机呼吸间隙里的自然眨眼，不是情绪表演。

## 必须保持不变的部分

以下内容不能变：

- 身体姿势
- 手的位置
- 腿的位置
- 尾巴轮廓
- 发型外轮廓
- 衣服结构
- 鞋袜
- 角色整体大小

允许微调的只有：

- 上下眼皮
- 少量眼部高光
- 极轻微的神情过渡

## 负面约束

不要出现：

- 重新设计角色脸
- 眼睛位置漂移
- 脸型变化
- 发型变化
- 耳朵大小变化
- 表情突然变成大笑
- 低幼卖萌感
- 油滑坏笑感
- 身体跟着乱动

## 输出格式

优先：

- 透明背景 PNG

命名建议：

- `feiling_idle_blink_half_v1.png`
- `feiling_idle_blink_closed_v1.png`

如果平台暂时不能稳定出透明 PNG，也至少要：

- 保持背景单独可去除
- 角色边缘干净
- 不要混入新的复杂背景

## 程序友好要求

- 两张图尺寸一致
- 角色在画面中的位置一致
- 脚底位置一致
- 头部位置一致
- 方便后面直接切帧

## 直接发给 agent 的任务描述

请基于当前已经确认的绯铃待机图，产出第一组最小眨眼素材。

要求：

- 只做待机眨眼
- 只输出 `blink_half` 和 `blink_closed`
- 严格保持角色设计、姿势、服装、发型、构图不变
- 只改眼睛闭合状态
- 风格保持当前绯铃待机图完全一致
- 不要重新创作角色
- 不要顺便修改表情和动作
- 目标是给桌宠程序做第一版 `blink` 动画接入
