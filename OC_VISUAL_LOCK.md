# 绯铃视觉锁定文档

## 这份文档的用途

这不是角色设定文档，也不是世界观文档。

这份文档只解决两件事：

- 一致性差
- 画出来不够 Q 版、不够像参考立绘

也就是说，这份文档专门负责把“绯铃到底长什么样”锁住。

## 核心原则

绯铃必须优先接近：

- Q 版桌宠
- 团子感
- 软糯感
- 白毛狐耳红瞳
- 桌面常驻的小型陪伴角色

而不是接近：

- 普通二次元少女立绘
- 手游角色卡面
- 高挑美少女
- 海报感插画

## 视觉总定位

绯铃的视觉方向是：

- 强 Q 版
- 低头身比
- 大头小身子
- 圆脸
- 大眼睛
- 轮廓柔软
- 重点在头部和表情

一句话概括：

她应该像“软乎乎的一团白狐桌宠”，而不是“正常比例的狐耳少女”。

## 头身比锁定

这是最重要的视觉约束。

必须满足：

- 头身比约 `2` 到 `2.5` 头身
- 头明显大于身体
- 身体短小
- 四肢短软

不要出现：

- 4 头身以上
- 正常人体比例
- 长腿
- 高挑
- 修长体态

如果头身比不对，就算颜色和耳朵对了，也会立刻不像参考图。

## 脸部锁定

绯铃的脸必须是：

- 圆脸
- 脸短
- 五官集中
- 眼睛非常大
- 鼻子存在感弱
- 嘴巴小

重点：

- 可爱感来自脸部占比高
- 不是靠成熟五官

不要出现：

- 尖下巴
- 长脸
- 偏成熟脸
- 写实脸
- 太“精致成人”的脸

## 眼睛锁定

眼睛是绯铃最重要的角色识别点之一。

要求：

- 大眼睛
- 红色或红橙渐变瞳
- 眼睛要有高光
- 眼型偏圆，不要细长锐利
- 看起来要“软”“亮”“灵”

不要出现：

- 冷艳细长眼
- 过于锐利的攻击感眼型
- 写实虹膜
- 眼睛过小

## 头发锁定

要求：

- 白色或奶白色长发
- 发量大
- 蓬松
- 发尾有明显卷度或空气感
- 整体像柔软团状云朵

重点：

- 发量必须足
- 头发轮廓要撑起角色体积感

不要出现：

- 贴头皮的头发
- 太稀薄
- 太利落太硬的发尾
- 写实分丝感过强

## 耳朵锁定

绯铃的狐耳必须是强识别点。

要求：

- 耳朵较大
- 明显立起
- 位置清楚
- 内耳偏浅粉或浅暖色
- 一眼能看出是狐耳，而不是普通发饰

不要出现：

- 耳朵太小
- 被头发完全遮住
- 画成猫耳但不够狐感

## 身体与轮廓锁定

要求：

- 身子短
- 肩窄
- 手脚小
- 整体像“团”
- 轮廓重心在上半身

重点：

- 可爱重心在头和上半身
- 不是站姿修长型角色

不要出现：

- 身体拉长
- 腿变成长腿
- 腰线太明显
- 少女立绘化

## 服装锁定

要求：

- 红白蓝配色
- 日系轻制服感
- 上半身服装有层次
- 袖子略带蓬感
- 可保留小红蝴蝶结作为记忆点

服装的目标不是复杂，而是：

- 一眼认得出
- 有角色识别性
- 缩小后也不乱

不要出现：

- 太华丽
- 太多飘带
- 太多复杂配件
- 战斗服
- 礼服
- 赛博装甲

## Q 版程度锁定

这一条必须反复强调给出图 agent：

绯铃是：

- Q 版桌宠

不是：

- 普通二次元角色立绘的“小号版本”

应满足：

- 头占画面重心
- 角色整体可爱重心在脸
- 身体只是承托
- 表情能一眼读懂

推荐关键词：

- 强 Q 版
- 超大头身比
- 团子感
- 软糯
- 桌宠感
- 表情包感

## 姿态锁定

第一批图建议统一成：

- 正面或微微偏正面
- 站立
- 重心稳定
- 构图固定

不要一开始就做：

- 大透视
- 大动态
- 跑动
- 旋转视角

因为这会直接破坏一致性。

## 参考图接近点

你给的参考立绘，真正要锁住的不是“白毛狐耳”这几个词，而是以下结构：

- 头非常大
- 脸非常圆
- 眼睛非常大
- 身体非常短
- 头发量感非常足
- 狐耳非常显眼
- 角色整体像一团软绵绵的小生物
- 不是细瘦型，而是团感型

如果出图 agent 没做到这些，就算配色正确，也还是会偏。

## 一致性锁定规则

所有后续状态图和动画图都必须保持：

- 同一头身比
- 同一脸型
- 同一发型
- 同一耳朵大小与位置
- 同一服装结构
- 同一配色逻辑
- 同一机位
- 同一角色比例

只能变化：

- 表情
- 轻微动作
- 手势
- 小幅姿态

不能大幅变化：

- 构图
- 视角
- 身体比例
- 脸型
- 服装款式

## 负面约束

这部分建议直接丢给出图 agent。

不要：

- 正常人体比例
- 4 头身以上
- 长腿
- 高挑
- 成熟女性脸
- 写实风
- 御姐风
- 海报感
- 大场景
- 复杂背景
- 手游角色立绘感
- 战斗系服装
- 写实狐耳
- 构图漂移
- 每张图像不同角色

## 直接给出图 agent 的视觉锁定版提示词

请严格按照“Q版桌宠”而不是“普通二次元立绘”来绘制绯铃。

必须满足：

- 2到2.5头身
- 头明显大于身体
- 圆脸
- 五官集中
- 超大眼睛
- 白色蓬松长发
- 大狐耳
- 红色渐变瞳
- 红白蓝轻制服
- 小红蝴蝶结
- 整体软糯、团子感、桌宠感
- 透明背景
- 正面站立
- 构图稳定
- 比例固定

绝对不要：

- 正常立绘比例
- 长腿
- 高挑
- 成熟脸
- 写实风
- 海报感
- 大动态
- 复杂背景
- 普通手游角色感

## 使用方法

后续出图时建议把提示拆成三层：

1. 固定视觉锁定词
2. 固定角色身份词
3. 最后只替换状态词

例如：

- 固定层：Q版桌宠、2到2.5头身、白毛狐耳红瞳、红白蓝制服、透明背景、固定机位
- 身份层：绯铃、白尾灵狐拟人、陪伴型桌宠、喜欢主人、微腹黑
- 状态层：待机 / 开心 / 思考 / 困惑 / 害羞 / 小得意

这样比每次重写一大段自由提示词更容易保持一致。
