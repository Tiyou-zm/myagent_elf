# 绯铃全身母版任务单

## 这份文档的用途

这份文档专门用于让出图 agent 基于当前已经较接近目标的版本，继续稳定产出绯铃的全身标准母版。

这不是世界观文档，也不是动作文档。

这份文档只负责一件事：

- 把绯铃的全身母版稳定下来，作为后续所有状态图和精灵图的统一参考。

## 当前阶段目标

当前不再继续自由探索新风格，也不再频繁更换服装主题。

当前唯一目标是：

- 固定绯铃的全身比例
- 固定绯铃的脸部和发型
- 固定绯铃的服装方向
- 产出一张能作为后续所有素材基准的全身母版

一句话：

现在要做的是“定母版”，不是“继续发散创作”。

## 当前角色方向

- 名字：绯铃
- 类型：白尾灵狐拟人桌宠
- 气质：安静、细心、很爱主人
- 个性补充：偶尔腹黑毒舌，但不是坏笑反派，不是轻佻表情包
- 产品定位：桌宠是主体，搜索只是她的一项能力

## 当前已经基本确定的视觉方向

- 强 Q 版
- 白毛
- 狐耳
- 红瞳
- 红白蓝制服系
- 团子感
- 桌宠感
- 日系萌系

## 这轮必须保留的内容

基于当前这张已经较接近目标的版本，后续生成时必须尽量保持以下内容不变：

- 脸型方向
- 眼睛大小和感觉
- 发型与发量感
- 狐耳大小和位置
- 红白蓝制服方向
- 小红蝴蝶结
- 整体安静软糯的气质

重点：

- 不要重新设计角色
- 不要重新换主题服装
- 不要重新探索完全不同风格

## 全身母版要求

请输出一张标准全身母版立绘：

- 透明背景
- 角色完整可见
- 脚部完整
- 尾巴完整
- 正面站立或极轻微正侧
- 构图居中
- 比例固定
- 适合后续扩展状态图和精灵图

## 脸部要求

当前脸部方向已经比较接近目标，后续不要大改，只允许微调。

要求：

- 保持软圆短脸
- 保持大眼睛
- 保持红色渐变瞳
- 保持可爱感和精致感

不要：

- 重新变成圆饼脸
- 变成长脸
- 变成成熟脸
- 变成坏笑脸
- 变成低幼表情包脸

## 表情要求

当前母版表情应以“默认待机脸”为主。

要求：

- 安静
- 柔和
- 略微有灵气
- 轻微嘴硬感可以有一点，但不要明显

不要：

- 夸张笑容
- 坏笑
- 轻佻
- 反派感
- 油滑感

一句话：

母版表情只要“安静可爱”，不要在母版阶段强塞太多性格戏。

## 头身比要求

要求：

- 保持 Q 版
- 约 2 到 2.5 头身
- 头明显大于身体
- 身子短
- 腿不要突然拉长

不要：

- 正常人体比例
- 高挑
- 长腿
- 一扩全身就从桌宠变成立绘少女

## 发型要求

要求：

- 白色蓬松长发
- 发量足
- 轮廓柔软
- 有一定空气感
- 整体偏顺，不要过度炸开

不要：

- 太贴头皮
- 太炸
- 太卷
- 太乱

一句话：

头发要蓬松，但要精致柔顺，不要毛绒玩偶化。

## 服装要求

当前服装方向已经比较合适，请保持：

- 红白蓝制服系
- 小助理感
- 简洁、利落、可缩小识别

建议保留：

- 深蓝裙子
- 红色小外套
- 领口蝴蝶结
- 简洁鞋子

不要：

- 女仆主题化
- 巫女化
- 和尚感
- 大量蕾丝
- 大量和风结构
- 夸张时装设计

## 身体比例要求

要求：

- 上半身仍然是视觉重点
- 手脚小巧
- 下半身不要喧宾夺主
- 尾巴和身体关系自然

不要：

- 腿长到抢视觉重心
- 尾巴像贴图一样飘在旁边
- 手脚比例乱跳

## 当前这轮最重要的任务

如果后续继续让 agent 调整，请只围绕以下 3 件事微调：

1. 全身比例是否稳定
2. 脸和发型是否保持
3. 服装和尾巴是否自然统一

不要同时做：

- 换服装
- 换主题
- 大改表情
- 大改脸型

## 当前最适合的工作方式

请按“母版精修”思路工作，而不是“新图重画”思路工作。

也就是说：

- 以当前最接近目标的全身图为基础
- 小步微调
- 不推翻重来

## 适合直接发给 agent 的任务描述

请基于当前已经较接近目标的绯铃版本，继续稳定她的全身标准母版。

要求：

- 保持现在的脸、眼睛、发型、狐耳、配色和整体气质
- 保持红白蓝制服系方向
- 输出完整全身、透明背景、正面站立的 Q 版母版立绘
- 比例固定，适合后续所有状态图和精灵图扩展
- 不要重设计角色
- 不要重新探索新服装主题
- 不要把她画成立绘少女
- 不要变成长腿高挑角色

目标不是生成一张新图，而是确定绯铃后续所有素材都要遵守的全身母版。
