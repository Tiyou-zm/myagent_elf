# 绯铃第一批状态图任务单

## 用途

这份文档用于让出图 agent 基于当前已经确认的绯铃全身母版，继续稳定产出第一批状态图。

这份文档只负责：

- 状态图

不负责：

- 新角色设计
- 新服装设计
- 动画分镜

## 当前前提

当前默认前提是：

- 绯铃的全身母版已经基本确定
- 后续所有状态图都必须严格沿用当前母版

重点：

- 不要重新设计脸
- 不要重新设计发型
- 不要重新设计服装
- 不要重新改头身比

## 当前阶段目标

产出绯铃第一批最常用、最适合桌宠 `v1` 的状态图。

这一批状态图主要用于：

- 桌宠待机显示
- 搜索中反馈
- 操作成功反馈
- 轻情绪表达

## 必须保持不变的内容

所有状态图都必须保持：

- 同一头身比
- 同一脸型
- 同一眼睛大小与颜色
- 同一发型与发量
- 同一狐耳大小和位置
- 同一服装结构
- 同一配色
- 同一尾巴结构
- 同一正面站立机位

只能变化：

- 表情
- 极小幅手势
- 极小幅姿态

不能变化：

- 角色设计
- 服装主题
- 机位
- 构图
- 比例

## 第一批状态图清单

请优先产出以下 5 张：

### 1. 待机 `idle`

用途：

- 默认常驻桌面显示

要求：

- 表情安静
- 柔和
- 不夸张
- 像在安静陪着主人

### 2. 开心 `happy`

用途：

- 搜索成功
- 完成任务
- 被主人点到后正反馈

要求：

- 比默认状态更亮一点
- 微笑
- 开心但不要过度兴奋

### 3. 思考 `thinking`

用途：

- 搜索中
- 整理中
- 等待结果中

要求：

- 神情认真
- 轻微思索感
- 不要夸张皱眉

### 4. 困惑 `confused`

用途：

- 没找到结果
- 遇到问题
- 需要主人补充信息

要求：

- 有一点不确定
- 但仍然可爱
- 不要画成智障或搞怪

### 5. 小腹黑得意 `tsundere_smirk`

用途：

- 主人又忘了东西
- 她猜对了
- 她轻轻拿捏主人时

要求：

- 要有一点“我就知道”的感觉
- 但不能坏笑
- 不能轻佻
- 不能反派感
- 应该是温柔偏心式的小得意

## 表情方向要求

绯铃的状态图不能走这几种错误方向：

- 表情包化
- 坏笑化
- 过度低幼
- 夸张搞怪
- 轻佻

正确方向应当是：

- 安静
- 细微
- 有灵气
- 很爱主人
- 偶尔有一点熟人式拿捏感

## 输出要求

- 透明背景
- 全身完整
- 构图统一
- 角色居中
- 分辨率与母版一致或更高
- 后续适合继续做桌宠界面显示

## 直接发给 agent 的任务描述

请基于当前已经确认的绯铃全身母版，输出第一批状态图。

要求：

- 严格沿用母版，不要重设计角色
- 保持同一脸、同一发型、同一服装、同一比例
- 只做表情和极小幅姿态变化
- 输出以下 5 张：
  - 待机
  - 开心
  - 思考
  - 困惑
  - 小腹黑得意

重点：

- 不要坏笑
- 不要搞怪
- 不要低幼表情包感
- 小腹黑得意必须是“温柔地拿捏主人”，不是“恶意坏笑”
